# Product Scope

## Audience

| Persona | Need | Product Support |
| --- | --- | --- |
| Curious learner | Understand ocean topics without academic friction. | Visual home screen, short summaries, daily facts. |
| Student | Explore reliable introductory marine science concepts. | Categories, topic cards, source links, quiz checks. |
| Educator or mentor | Preview a structured learning companion. | Trackable topic taxonomy and source-aware content. |
| Ocean enthusiast | Browse engaging ocean knowledge casually. | Search, filters, imagery, quick interactions. |

## MVP Features

| Priority | Feature | Status |
| --- | --- | --- |
| Must | Expo React Native app shell | Done |
| Must | Ocean-themed home screen | Done |
| Must | Local typed knowledge base | Done |
| Must | Explore screen with search and filters | Done |
| Must | Interactive quiz check | Done |
| Should | Saved articles | Done |
| Should | Article detail route | Done |
| Should | Ocean Atlas screen | Done |
| Later | Backend sync, auth, user profiles, community content | Not Started |

## Success Criteria

- A new visitor can understand the app purpose within the first screen.
- Users can browse, search, and expand ocean topics without setup.
- Seed content is source-aware and safe for a public repository.
- The interface feels modern, image-rich, mobile-first, and ocean-themed.
- TypeScript and lint checks pass before release review.
- Core knowledge-base utilities have automated tests.

## Out of Scope for First Release Slice

- User accounts or authentication.
- Live backend content management.
- Paid APIs or private datasets.
- Social/community features.
- Production app-store release packaging.
